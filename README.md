# SafeSpace Drill Mode

This branch contains the Figma-exported drill mode UI for the SafeSpace mobile game.

## Run locally

1. Install dependencies with `npm install`.
2. Start the dev server with `npm run dev`.
3. Build the app with `npm run build`.

## Notes

- The repo has been cleaned up so the Figma app uses the canonical root files: `index.html`, `package.json`, and `README.md`.
