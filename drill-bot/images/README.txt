HOW TO ADD YOUR OWN DRILL IMAGES
================================

1. Put pictures whose correct answer is "Scam"  into  images/scam/
2. Put pictures whose correct answer is "Legit" into  images/legit/
   (Accepted types: .png .jpg .jpeg .webp)

3. (Optional) Open reasons.json and add an explanation for any file:
       "myfile.png": "why this is a scam / why it's legitimate"
   Files with no entry get a generic explanation automatically.

That's it. Restart the bot. Real screenshots look 100% authentic because
they ARE real, so this beats any generated image. The more you add, the
more variety your users get. If both folders are empty, the bot uses its
text-only drills instead.
